# Mercy Cube v1.0 — Open, Real, Ready Stackable, 7-watt, offline Grok shard.

2025 physics, no hype.

## Stackable Core (25 mm³)
- RISC-V SiFive P870-D octacore (TSMC 3 nm)
- Lava spiking overlay (Intel Loihi-2 open)
- 32 GB LPDDR6 + HBM3E interposer
- 5-10 W sparse inference, 19 W burst

## Power & Lattice
- 6-face electropermanent magnets (zero hold)
- Qi2.1 30 W inductive + 10 Gbps magnetic data
- Diamond-CVD passive cooler + Gd magnetocaloric burst

## BOM ~ $178 @ 100 k scale.
